# 06. 페이지 구조 & UI 설계

## 라우팅

```
/                           # 랜딩 페이지 (비로그인용)
/login                      # 로그인
/signup                     # 회원가입

/dashboard                  # 대시보드 (로그인 메인)
/upload                     # 사진 업로드 + 모드 선택
/post/[postId]              # 글 상세/편집/영상 생성
/mypage                     # 마이페이지 (탭: 블로그 / 다이어리)
/mypage/videos              # 생성한 영상 목록
/settings                   # 설정 (사용량, 플랜 등)
```

## 공통 레이아웃

### 사이드바 (`components/layout/Sidebar.tsx`)

시안 기반 좌측 고정 사이드바:

```
┌─────────────────┐
│ 🌿 PhotoLog     │  ← 로고 + 브랜드명
├─────────────────┤
│ 📊 대시보드     │
│ ✏️ 새 글 작성   │
│ 📚 마이페이지   │
│ 🎬 내 영상      │
│ 🔔 알림         │
│ ⚙️ 설정         │
├─────────────────┤
│ [프로필 영역]   │  ← 하단 고정
└─────────────────┘
```

**스펙:**
- 너비: 240px (`w-60`)
- 배경: #FFFFFF
- 활성 메뉴: `bg-primary-100 text-primary-500` + 좌측 4px primary 인디케이터
- 호버: `bg-background-subtle`
- 아이콘: lucide-react

### 헤더 (`components/layout/Header.tsx`)

```
인녕하세요, [닉네임] 👋          [날짜선택] [알림🔔] [프로필]
오늘도 좋은 콘텐츠 만들어보세요
```

- 좌측: 인사말 + 동기부여 메시지
- 우측: 날짜 선택, 알림 벨, 프로필 아바타

## 핵심 화면별 설계

### 랜딩 페이지 (`/`)

비로그인 사용자용. 별도 레이아웃(사이드바 없음).

**섹션 구성:**
1. **Hero**
   - 헤드라인: "사진만 올리면, 블로그 글과 숏폼 영상이 짜잔"
   - 서브: "EXIF·GPS·OCR을 분석해 진짜 내 경험 그대로 콘텐츠로"
   - CTA: `무료로 시작하기` (Primary) / `데모 보기` (Outline)
   - 시각: 사진 → 글 → 영상 변환 GIF

2. **Before / After 비교**
   - 좌: "사진 587장 갤러리에 방치"
   - 우: "PhotoLog: 사진 던지면 30초 자동"

3. **핵심 기능 3카드**
   - 📸 사진이 곧 데이터 (EXIF·GPS·OCR)
   - ✍️ 두 가지 모드 (블로그/다이어리)
   - 🎬 영상까지 한 번에

4. **How it works (4-Step)**
   - 사진 업로드 → 모드 선택 → AI 생성 → 발행/영상

5. **결과물 예시**
   - 실제 생성 블로그 글 1개 + 숏폼 영상 1개

6. **가격**
   - Free 플랜 강조

7. **FAQ** (Accordion)

8. **Footer**

### 대시보드 (`/dashboard`)

시안과 동일한 4-카드 + 차트 레이아웃:

```
┌──────────┬──────────┬──────────┬──────────┐
│ 이번 달  │ 누적 글  │ 영상 생성│ 평균 시간│
│ 작성 글  │ 총 개수  │ 횟수     │ 절약     │
│ 12편     │ 47편     │ 8편      │ 24시간   │
│ +12.4%   │ +8.1%    │ +0.8%    │ +5.3%    │
└──────────┴──────────┴──────────┴──────────┘

┌─────────────────────────┬───────────────────┐
│ 작성 추이 (라인 차트)   │ 모드 비율         │
│  Recharts LineChart     │ Recharts          │
│                         │ DonutChart        │
│  블로그/다이어리 추이   │ 블로그 60%        │
│                         │ 다이어리 40%      │
└─────────────────────────┴───────────────────┘

┌─────────────────────────┬───────────────────┐
│ 최근 작성 글            │ 오늘의 알림       │
│ • 풍차로 가는길 후기    │ • 영상 렌더 완료  │
│ • 5월 3일 일요일        │ • 월 사용량 50%   │
│ • 성수동 카페 투어      │                   │
└─────────────────────────┴───────────────────┘
                                  [일러스트]
```

**컴포넌트:**
- `StatCard` — 좌측 4px 컬러 보더 + 아이콘 박스 + 숫자 + 변화율 뱃지
- `TrendChart` — Recharts LineChart, 컬러는 primary-300
- `ModeRatioChart` — Recharts PieChart (도넛), 블로그=primary, 다이어리=lavender
- `RecentPostsList` — 최근 5개 글
- `NotificationList` — 알림 카드

### 사진 업로드 (`/upload`)

Step별 큰 카드 형태로:

```
[Step 1] 사진을 올려주세요
  ┌────────────────────────────┐
  │  🖼️ 드래그 앤 드롭         │
  │     또는 파일 선택         │
  │  (최소 3장 ~ 최대 30장)    │
  └────────────────────────────┘
  
  업로드된 사진 미리보기 (시간순 자동 정렬)
  ┌────┬────┬────┬────┬────┐
  │ 1  │ 2  │ 3  │ 4  │ 5  │ ...
  └────┴────┴────┴────┴────┘

[Step 2] 어떤 형태로 만들까요?
  ┌─────────────┐  ┌─────────────┐
  │ 📝 블로그   │  │ 📔 다이어리 │
  │ 부업·발행용 │  │ 개인 기록용 │
  └─────────────┘  └─────────────┘

[Step 3] 톤을 선택해주세요
  Select 컴포넌트
  [ 정보형 / 감성형 / 리뷰형 ]
  
  제목 (선택)
  [Input — 비워두면 AI가 자동 생성]

[Step 4]
  [ ✨ AI로 생성하기 ]  ← 큰 Primary 버튼

생성 중에는 진행 상태 표시:
✓ EXIF 추출 중...
✓ 위치 정보 매칭 중...
✓ OCR 분석 중...
○ AI 글 생성 중... (현재)
```

**컴포넌트:**
- `PhotoDropzone` — react-dropzone 활용
- `PhotoPreview` — 시간순 정렬된 썸네일 그리드
- `ModeSelector` — 큰 2-카드 선택 UI
- `ToneSelector` — Select 컴포넌트
- `GenerationProgress` — 단계별 진행 표시


**컴포넌트:**
- `SceneCard` — 단락 1개 단위, 사진 + 본문 + 자막
- `PostEditor` — 전체 글 편집 컨테이너
- `PostSidebar` — 우측 액션 패널

## 모바일 대응

- 사이드바 → `Sheet` 컴포넌트로 변환 (햄버거 메뉴)
- 그리드 → 1단으로 축소
- Step UI → 세로로 펼쳐짐
- 글 결과 화면 → 사이드바를 하단 고정 액션 바로

```typescript
// Tailwind 반응형
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
```

## 빈 상태(Empty State) UI

각 화면에 빈 상태 디자인:
- 일러스트 (Storyset)
- 한 줄 안내 메시지
- CTA 버튼

예: 마이페이지 빈 상태
```
[식물 일러스트]
아직 작성된 글이 없어요
첫 번째 글을 만들어볼까요?
[ 사진 업로드하기 ]
```
