# 01. 프로젝트 개요

## 한 줄 정의

사진을 업로드하면 EXIF·GPS·OCR·Vision AI가 사실 데이터를 추출하여 **블로그형 글** 또는 **다이어리형 글**을 자동 생성하고, 사용자가 선택 시 단락 단위 JSON으로 변환해 **숏폼 영상까지** 자동 생성하는 SaaS.

## 문제 정의

- 폰에 쌓인 수백 장의 사진은 끝내 블로그·SNS에 올라가지 못하고 삭제됨
- 기존 AI 글쓰기는 누가 써도 같은 글 → 저품질 분류·애드센스 거절
- 글과 영상을 따로 만드는 이중 작업 부담 (글: ChatGPT, 영상: CapCut)
- 생성형 AI 영상(Sora·Runway·Kling)은 5~20초 한계, 비싸고 일관성 부족

## 솔루션 — 4-Layer 데이터 분석 엔진

| Layer | 데이터 | 활용 |
|---|---|---|
| 시간 | EXIF DateTime | 동선 자동 정렬, 시간순 스토리 |
| 위치 | EXIF GPS | 역지오코딩 → 상호명·주소·영업시간 |
| 텍스트 | OCR | 메뉴판·간판·가격표 → 본문 삽입 |
| 시각 | Vision AI | 사진 픽셀 단위 해석, 분위기·디테일 묘사 |

→ 4-Layer 통합 → JSON 구조화 → 블로그 글 + Remotion 영상 동시 출력

## 차별화 포인트

### 1. 사실 기반 AI 글
EXIF·GPS는 위조 불가능. 구글 E-E-A-T(경험성) 자동 충족. 다른 AI 글 도구가 흉내 낼 수 없는 영역.

### 2. 글 → 영상 동시 출력
같은 JSON 데이터로 블로그/다이어리/숏폼 모두 생성. Remotion 기반 영상 자동 렌더링, 길이 제한 무.

### 3. 이중 모드
- **블로그 모드**: 부업·콘텐츠 발행용 (SEO·정보형)
- **다이어리 모드**: 개인 기록용 (1인칭·감성)

같은 사진 데이터, 모드만 다르게 선택하면 출력 톤·구조가 자동으로 바뀜.

## 시장 포지셔닝

| 카테고리 | 대표 서비스 | 한계 |
|---|---|---|
| AI 글쓰기 | 가제트AI, ChatGPT | 글만, 사실성 0 |
| AI 영상 생성 | Sora, Runway, Kling | 5~20초, 가짜 픽셀 |
| 영상 편집 SaaS | CapCut, Pictory | 사람이 직접 작업 |
| **PhotoLog** | — | **사진 → 글+영상, 빈 카테고리** |

## 타겟 사용자

### Phase 1 (MVP)
**부업·콘텐츠 자동화가 필요한 개인** — 블로그·숏폼으로 부업하려는 사람, 자영업자, 체험단 인플루언서, 1인 마케터

### Phase 2 (확장)
**개인 기록자** — 다이어리 모드 활성화, 라이프로그 영역 확장
